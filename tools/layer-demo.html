② 딸기 정식기 P–I 곡선 — 최적 작동점 고정

모델식  Pn = (α·I·Pmax) / (α·I + Pmax) − Rd 직각쌍곡선 광반응곡선(rectangular hyperbola) · I = PAR(광량), Pmax = Pmax0·f(T)·f(CO₂) (온도·CO₂ 반응으로 변조), Rd = 암호흡. 계수 Pmax0=26·α=0.06·Rd=1.5, Topt=21.5°C, CO₂ 포화 ~1300ppm. 식 형태는 식물생리 문헌 표준이며, 계수는 딸기 정식기 문헌 범위 내 추정치(실측 미피팅).
제어 전(현재) 제어 후(최적 온도+CO₂) · 초록 세로선=정식기 최적 작동점(고정) · 주황선=예산 내 달성 · ◆=포화 무릎

④ 예상 시간 응답 (1차 지연) — 온도·CO₂가 목표로

현재값에서 선택 구동기로 목표(온도 21.5°C·CO₂ 목표)에 1차 지연으로 수렴. 천창 환기 ~0.15°C/min·능동냉방 ~0.4°C/min·CO₂는 환기율(ACH) 기반 감쇠. (점선=목표)
